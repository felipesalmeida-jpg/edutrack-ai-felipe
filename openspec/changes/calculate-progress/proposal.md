# Change: calculate-progress

## Why
O sistema precisa calcular a porcentagem de progresso de uma disciplina
baseada nas tarefas concluídas versus o total de tarefas.

## What Changes
- Criação do script scripts/calculate_progress.py que recebe uma lista
de tarefas e retorna um JSON com a porcentagem de progresso

## Impact
- Sem impacto no banco de dados
- Sem impacto no frontend neste momento
